A dropdown menu component triggered by a button, displaying a list of actions with keyboard navigation, submenus, checkbox items, and radio groups.

This component is based on Kobalte's `DropdownMenu` and follows Base-UI's `Menu` naming conventions: `Menu.Root` wraps the menu, `Menu.Trigger` is the button that opens it, and `Menu.Popup` holds the content. Submenus use `Menu.SubmenuRoot`, `Menu.SubmenuTrigger`, and `Menu.SubmenuPopup`.

For a horizontal menubar with multiple dropdown menus, see [Menubar](/components/menubar/).
