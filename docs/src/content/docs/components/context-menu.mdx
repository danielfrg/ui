A context menu component that appears at the pointer position when the user right-clicks or long-presses on a trigger area.

This component is based on Kobalte's `ContextMenu`. `ContextMenu.Root` wraps the menu, `ContextMenu.Trigger` defines the right-click area (renders a `div`), and `ContextMenu.Popup` holds the content. The menu opens at the cursor position rather than anchored to a trigger element.

For a button-triggered dropdown menu, see [Menu](/components/menu/). For a horizontal menubar, see [Menubar](/components/menubar/).
