Unstyled and accessible SolidJS components mostly forked from [Kobalte](https://kobalte.dev) and others migrated from [Base UI](https://base-ui.com/).

# Quick start

Install the package:

```bash
npm install @danielfrg/ui
```

Import the components you need:

```tsx
import { Button } from "@danielfrg/ui"
```

Use the components:

```tsx
<Button>Click me</Button>
```
