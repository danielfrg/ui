Styled and accessible SolidJS components with a shadcn-inspired Tailwind theme. Built on top of headless primitives from `@danielfrg/ui-core` (forked from [Kobalte](https://kobalte.dev) and [Base UI](https://base-ui.com/)).

# Quick start

Install the styled components:

```bash
npm install @danielfrg/ui
```

Add the theme to your CSS:

```css
@import "tailwindcss";
@import "@danielfrg/ui/theme.css";
```

Import and use:

```tsx
import { Button } from "@danielfrg/ui/button"

;<Button variant="outline">Click me</Button>
```

For headless/unstyled primitives only:

```bash
npm install @danielfrg/ui-core
```

```tsx
import { Button } from "@danielfrg/ui-core/button"
```
