Example layouts demonstrating how to compose `@danielfrg/ui` components into real-world application interfaces. These are SolidJS translations of the [Radix Themes](https://www.radix-ui.com/themes) example layouts, built using headless components from this library.

## Available Examples

### [Dashboard](/examples/dashboard/)

A comprehensive business dashboard featuring team management, notifications, pricing tiers, sign-up forms, credit cards, invoices, financial KPIs, activity feeds, and to-do lists. Demonstrates: **Avatar**, **Button**, **Checkbox**, **Switch**, **Separator**, **Menu**, **Input**.

### [E-commerce](/examples/ecommerce/)

A full e-commerce interface with product cards, shopping cart, filters, delivery tracking, order management, bookmarks, product editing, shipment tracking, promotional banners, and customer analytics. Demonstrates: **Button**, **Separator**, **Switch**, **Select** (native).

### [Music App](/examples/music/)

A desktop music player with queue management, equalizer, album browsing, now-playing bar, settings, membership pricing, user profiles, playlist creation, and lyrics display. Demonstrates: **Avatar**, **Button**, **Checkbox**, **Switch**, **Separator**, **Slider** (native).
